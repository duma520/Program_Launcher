## 项目概述

程序启动器是一款基于PyQt5开发的桌面应用程序，旨在为用户提供一个高效、美观且可定制的程序启动管理工具。它不仅仅是一个简单的快捷方式集合，更是一个强大的工作流管理工具。

### 设计理念
- **直观易用**：简洁的界面设计，适合所有技术水平用户
- **高度可定制**：支持分组管理、自定义图标、启动参数等
- **高效工作**：快速搜索、收藏功能、批量操作提升效率
- **跨平台兼容**：主要支持Windows，部分功能兼容其他操作系统

### 适用人群
- **普通用户**：管理日常应用程序
- **办公人员**：快速启动工作相关软件套件
- **开发者**：管理开发工具链和常用命令
- **设计师**：组织设计软件和资源
- **系统管理员**：管理服务器工具和脚本
- **游戏玩家**：整理游戏启动器

## 安装与运行

### 系统要求
- **操作系统**：Windows 7/10/11 (推荐Windows 10+)
- **Python版本**：3.7+
- **内存**：至少512MB可用内存
- **磁盘空间**：至少50MB可用空间

### 安装方法

#### 方法一：直接运行（推荐非技术用户）
1. 从发布页面下载打包好的可执行文件
2. 双击`ProgramLauncher.exe`运行
3. 首次运行会自动创建必要文件和数据库

#### 方法二：从源代码运行（开发者）
```bash
git clone https://github.com/duma520/ProgramLauncher.git
cd ProgramLauncher
pip install -r requirements.txt
python Program_Launcher.py
```

### 首次运行配置
1. 程序会自动创建`launcher.db`数据库文件
2. 生成`icons`目录用于存储图标
3. 创建`backups`目录用于数据库备份

## 基础使用指南

### 1. 添加第一个程序
1. 点击"添加分组"按钮创建一个分组（如"办公软件"）
2. 选择该分组标签页
3. 点击"添加按钮"
4. 填写程序信息：
   - 名称：Word 2025
   - 路径：浏览选择Word.exe
   - (可选)参数：/n "新文档.docx"
5. 点击"保存"

### 2. 启动程序
- 直接点击对应按钮即可启动程序
- 右键按钮可查看详细信息和更多选项

### 3. 基本管理
- **重命名**：右键按钮→编辑
- **删除**：右键按钮→删除
- **移动**：右键按钮→移动到→选择目标分组

### 4. 使用搜索
1. 在顶部搜索框输入关键词
2. 实时显示匹配结果
3. 点击"搜索"按钮查看完整结果

## 高级功能详解

### 1. 管理员权限运行
在编辑按钮时勾选"以管理员权限运行"，适合需要提升权限的程序如：
- 注册表编辑器
- 磁盘管理工具
- 某些开发工具

### 2. 工作目录设置
指定程序启动时的初始目录，例如：
- 代码编辑器：设置为项目目录
- 命令行工具：设置为脚本所在目录
- 资源管理器：设置为常用目录

### 3. 启动参数
为程序添加启动参数，例如：
- Chrome：`--incognito` (无痕模式)
- Photoshop：`-new` (新建文档)
- Word：`/n "模板.dotx"` (基于模板新建)

### 4. 图标管理
- **自动提取**：EXE文件自动提取图标
- **自定义图标**：支持ICO/PNG/JPG格式
- **动态生成**：为无图标程序生成文字图标

### 5. 批量操作模式
1. 点击"批量操作"进入模式
2. 点击选择多个按钮（变蓝表示选中）
3. 右键任一选中按钮选择操作：
   - 批量移动到其他分组
   - 批量删除

### 6. 数据库备份与恢复
- 自动每小时备份一次
- 手动备份：关闭程序时自动执行
- 备份文件存储在`backups`目录
- 恢复方法：用备份文件替换`launcher.db`

## 专业用户指南

### 1. 数据库结构
程序使用SQLite数据库存储配置，主要表结构：

#### groups表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | TEXT | 分组名称 |
| position | INTEGER | 排序位置 |
| is_favorite | INTEGER | 是否收藏 |

#### buttons表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| group_id | INTEGER | 所属分组ID |
| name | TEXT | 按钮名称 |
| path | TEXT | 程序路径 |
| arguments | TEXT | 启动参数 |
| working_dir | TEXT | 工作目录 |
| run_as_admin | INTEGER | 管理员权限 |
| icon_path | TEXT | 图标路径 |
| position | INTEGER | 排序位置 |
| is_favorite | INTEGER | 是否收藏 |

### 2. 高级配置
通过修改`launcher.db`可以直接调整配置：
- 修改分组顺序：调整`position`值
- 批量编辑：使用SQL语句更新

### 3. 命令行参数
支持以下启动参数：
- `--reset`：重置窗口位置
- `--clean`：清除所有数据（慎用）
- `--db-path=<路径>`：指定数据库路径

### 4. 开发扩展
可以通过继承主要类实现功能扩展：
```python
class CustomLauncher(MainWindow):
    def __init__(self):
        super().__init__()
        # 添加自定义功能
```

## 常见问题解答

### 1. 程序无法启动
- 检查Python环境是否安装正确
- 确保所有依赖库已安装
- 查看是否有错误提示

### 2. 图标不显示
- 检查图标文件路径是否正确
- 确认图标文件未被占用
- 尝试重新设置图标

### 3. 管理员权限无效
- 确认用户账户有管理员权限
- 检查UAC设置是否过高
- 尝试以管理员身份运行启动器本身

### 4. 数据库损坏
- 从`backups`目录恢复最新备份
- 或删除`launcher.db`重新开始

### 5. 搜索不工作
- 确认输入了有效关键词
- 检查数据库是否正常加载
- 尝试重启程序

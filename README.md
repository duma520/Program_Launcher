## 项目概述

程序启动器是一款基于PyQt5开发的桌面应用程序，旨在帮助用户高效管理和快速启动各种程序、文件和工具。无论您是普通用户、IT专业人士还是系统管理员，都能从中受益。

### 适用人群
- **普通用户**：简化日常程序启动流程
- **办公人员**：快速访问常用办公软件和文档
- **开发者**：管理开发工具链和项目文件
- **设计师**：组织设计软件和资源文件
- **系统管理员**：批量管理系统工具和脚本
- **教育工作者**：整理教学软件和资料

## 安装指南

### Windows用户
1. 下载最新版本的可执行文件(.exe)
2. 双击运行安装程序
3. 按照向导完成安装
4. 桌面和开始菜单将创建快捷方式

### macOS用户
1. 下载.dmg镜像文件
2. 打开镜像并将应用拖到Applications文件夹
3. 首次运行时在系统偏好设置中允许运行

### Linux用户
1. 下载AppImage文件
2. 给文件添加可执行权限：`chmod +x ProgramLauncher.AppImage`
3. 双击运行或通过终端启动

### Python开发者
1. 确保安装Python 3.8+
2. 安装依赖：`pip install -r requirements.txt`
3. 运行主程序：`python Program_Launcher.py`

## 基础使用

### 1. 添加第一个分组
1. 点击"添加分组"按钮
2. 输入分组名称(如"办公软件")
3. 勾选"收藏分组"可将其置顶
4. 点击"保存"

### 2. 添加程序按钮
1. 选择目标分组标签页
2. 点击"添加按钮"
3. 填写程序信息：
   - 名称：显示在按钮上的文字
   - 路径：程序可执行文件路径
   - 参数：启动时传递的参数(可选)
   - 工作目录：程序运行时的工作目录(可选)
4. 设置图标：
   - 自动提取EXE文件图标
   - 手动选择图标文件(.ico/.png)
5. 点击"保存"

### 3. 启动程序
- 单击按钮即可启动对应程序
- 右键按钮可查看更多选项

### 4. 搜索功能
1. 在顶部搜索框输入关键词
2. 系统将实时显示匹配结果
3. 点击"搜索"按钮查看完整结果列表

## 高级功能

### 1. 批量操作模式
1. 点击"批量操作"按钮进入批量模式
2. 单击选择多个按钮(选中的按钮会高亮显示)
3. 右键任意选中按钮选择操作：
   - 批量移动到其他分组
   - 批量删除

### 2. 管理员权限运行
在编辑按钮时勾选"以管理员权限运行"，适合需要提升权限的程序如：
- 注册表编辑器
- 磁盘管理工具
- 系统配置程序

### 3. 启动参数示例
- `notepad.exe "C:\notes.txt"` - 用记事本打开指定文件
- `chrome.exe --incognito` - 以无痕模式启动Chrome
- `python script.py --debug` - 带调试参数运行Python脚本

### 4. 工作目录设置
- 对于需要访问特定文件的程序，设置工作目录可确保程序正确找到相关文件
- 示例：将Python脚本的工作目录设置为脚本所在文件夹

## 常见问题

### Q1: 程序无法启动某些应用
- 检查路径是否正确
- 确认有足够的权限运行目标程序
- 尝试在命令提示符中手动运行相同命令测试

### Q2: 图标显示不正确
- 确认图标文件未被移动或删除
- 尝试重新设置图标
- 检查图标文件格式是否受支持(.ico最佳)

### Q3: 如何迁移设置到新电脑
1. 备份`launcher.db`和`icons`文件夹
2. 在新电脑上安装程序启动器
3. 替换新电脑上的数据库和图标文件

### Q4: 搜索功能找不到某些项目
- 确认名称拼写正确
- 尝试使用拼音首字母搜索
- 检查是否在正确的分组中搜索

## 技术细节

### 1. 架构设计
- 前端：PyQt5实现响应式界面
- 后端：SQLite数据库存储配置
- 图标处理：Pillow+Win32API

### 2. 数据存储
- 使用SQLite数据库(`launcher.db`)
- 图标存储在`icons`子目录
- 备份保存在`backups`目录

### 3. 兼容性
- 支持Windows 10/11
- 兼容macOS 12+

### 4. 性能优化

## 开发者指南

### 1. 构建环境
```bash
# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/macOS
```

## 项目概述

### 什么是程序启动器？
程序启动器是一款基于PyQt5开发的桌面应用程序，旨在为用户提供一个高效、美观的程序和文件管理工具。它允许您：
- 分类组织常用程序和文件
- 快速启动应用程序
- 自定义分组和图标
- 支持管理员权限运行
- 提供强大的搜索功能

### 适用人群
- **普通用户**：简化日常程序访问
- **办公人员**：快速访问工作相关软件
- **开发人员**：管理开发工具链
- **系统管理员**：批量管理系统工具
- **设计师**：组织创意软件
- **教育工作者**：整理教学资源

### 核心优势
- 直观的图形界面
- 高度可定制化
- 轻量级设计
- 跨平台兼容性(主要支持Windows)
- 数据安全备份

## 功能特点

### 1. 程序管理
- **分组系统**：创建逻辑分组管理不同类别程序
  *示例*：创建"办公软件"、"开发工具"、"游戏"等分组
- **快速启动**：一键启动应用程序或打开文件夹
- **参数支持**：可为程序设置启动参数

### 2. 界面定制
- **马卡龙色系**：提供柔和美观的界面配色
- **图标支持**：
  - 自动提取EXE文件图标
  - 支持自定义ICO/PNG图标
  - 动态生成文字图标
- **布局调整**：自由调整按钮位置和分组顺序

### 3. 高级功能
- **管理员权限**：可选择以管理员身份运行程序
- **工作目录**：为每个程序设置独立工作目录
- **批量操作**：支持批量移动、删除按钮
- **收藏系统**：标记常用项目和分组

### 4. 搜索与组织
- **即时搜索**：支持名称和拼音首字母搜索
- **智能排序**：收藏项目自动置顶
- **结果筛选**：分类显示搜索结果

### 5. 数据安全
- **自动备份**：每5分钟自动备份数据库
- **备份管理**：保留最近24个备份版本
- **数据恢复**：可从备份手动恢复数据

## 安装指南

### Windows安装
1. **直接运行版**：
   - 下载提供的EXE文件
   - 双击即可运行，无需安装

2. **Python环境运行**：
   ```bash
   pip install -r requirements.txt
   python Program_Launcher.py
   ```

### 系统要求
- 操作系统：Windows 7/10/11 (推荐Windows 10+)
- 内存：至少512MB可用内存
- 存储：50MB可用空间

### 首次运行配置
1. 程序会自动创建数据库和图标目录
2. 建议立即创建第一个分组
3. 添加常用程序快捷方式

## 基础使用

### 1. 创建第一个分组
1. 点击"添加分组"按钮
2. 输入分组名称(如"工作软件")
3. 勾选"收藏分组"可将其置顶
4. 点击"保存"

### 2. 添加程序快捷方式
1. 选择目标分组
2. 点击"添加按钮"
3. 填写程序信息：
   - 名称：显示名称(如"Word")
   - 路径：浏览选择.exe文件
   - (可选)设置启动参数和工作目录
4. 点击"保存"

### 3. 启动程序
- 单击分组中的按钮即可启动对应程序
- 右键按钮可查看更多选项

### 4. 基本搜索
1. 在顶部搜索框输入关键词
2. 系统会实时筛选匹配项
3. 点击"搜索"按钮查看完整结果

## 高级功能

### 1. 批量操作模式
1. 点击"批量操作"按钮进入该模式
2. 单击选择多个按钮(背景变蓝)
3. 右键任一选中按钮选择：
   - 批量移动到其他分组
   - 批量删除

*专业提示*：适合IT管理员批量部署办公软件快捷方式

### 2. 管理员权限运行
在编辑按钮时勾选：
- "以管理员权限运行"
- 适用于需要提升权限的程序如：
  - 注册表编辑器
  - 磁盘管理工具
  - 网络配置工具

### 3. 工作目录设置
- 指定程序启动时的初始目录
- 对以下情况特别有用：
  - 开发IDE项目
  - 文档处理软件默认文件夹
  - 需要特定运行环境的程序

### 4. 图标管理技巧
- **自动提取**：EXE文件自动获取图标
- **目录图标**：自动使用文件夹图标
- **自定义图标**：
  1. 准备ICO/PNG文件(推荐64x64)
  2. 在编辑界面点击"设置图标"
  3. 选择图标文件

*设计建议*：保持图标风格一致可获得更好视觉效果

## 常见问题解答

### 1. 程序无法启动某些应用
- 检查路径是否正确
- 验证目标程序是否已安装
- 尝试以管理员身份运行

### 2. 图标显示不正常
- 确认图标文件未被移动
- 尝试重新设置图标
- 检查图标文件格式(推荐ICO)

### 3. 搜索功能不工作
- 确保输入了有效关键词
- 检查数据库是否正常加载
- 尝试重启程序

### 4. 数据丢失问题
- 检查backups目录下的自动备份
- 手动备份重要配置
- 避免直接编辑数据库文件

### 5. 界面显示异常
- 尝试调整窗口大小
- 检查系统DPI设置
- 确保显卡驱动最新

## 技术细节

### 1. 系统架构
- **前端**：PyQt5框架
- **数据存储**：SQLite数据库
- **图标处理**：Pillow+Win32API

### 2. 数据库结构
- **groups表**：存储分组信息
  ```sql
  CREATE TABLE groups (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    position INTEGER,
    is_favorite INTEGER
  )
  ```
- **buttons表**：存储程序快捷方式
  ```sql
  CREATE TABLE buttons (
    id INTEGER PRIMARY KEY,
    group_id INTEGER,
    name TEXT NOT NULL,
    path TEXT NOT NULL,
    arguments TEXT,
    working_dir TEXT,
    run_as_admin INTEGER,
    icon_path TEXT,
    position INTEGER,
    is_favorite INTEGER,
    FOREIGN KEY(group_id) REFERENCES groups(id)
  )
  ```

### 3. 关键技术
- **图标提取**：使用多种Win32 API方法
- **权限提升**：通过ShellExecuteEx实现
- **搜索算法**：支持拼音首字母匹配

### 4. 性能优化
- 延迟加载资源
- 后台备份机制
- 智能内存管理
